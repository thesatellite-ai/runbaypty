# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good faith, no harassment of any kind. Report conduct issues privately via GitHub's report abuse tooling or to the maintainer (@khanakia). Maintainers may remove, edit, or reject contributions and contributors that violate this standard.
